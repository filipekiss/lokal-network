Lokal Network

Zero config localhost domains

What is Lokal Network?

Lokal Network is a free, simple to use, zero configuration localhost domain provider. It allows you to use named domains for development projects runnings on local networks, so you can test things like CORS policies, OAuth integrations and other things that might require you use a proper domain name instead of localhost.

How do I use it?

First, ensure you have a service running on your machine (for example, NGINX or some Express.js application). Let's say you use localhost:3000 to access this server. Just point your browser to anything.lokal.network:3000 and you're done. That's really it.

Pro-tip: anything actually means anything. You can point your browser to my-awesome-server.lokal.network and the result will be the same.

But why? (and how)

I actually made this to show a few friends how A records work when it comes to DNS. Except for the apex domain, that is, lokal.network, the only DNS record is a wildcard that matches any subdomain and serves things from 127.0.0.1.

Is it safe?

Yes, absolutely. There are no servers for Lokal Network. This page is hosted on Github Pages and is pure HTML and CSS. And when you access a subdomain that shows you your local server, Lokal Network only tells you browser (or Postman or whatever it is that's requesting something) to look to your local server. We collect no data, intercept no traffic and log nothing.
